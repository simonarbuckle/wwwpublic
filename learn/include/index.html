What you will learn..

Core Layer

  1. Write a Class Definition
  2. Load a Class Definition from an HTML File
  3. Write a Singleton, Load it and Use it
  4. Implement Inheritance
  5. Manage Your Dependencies
  6. Discover Helpers Libraries
  7. Unit Test Your Code Using Attester

Template Engine Layer

  1. Create a Template File
  2. Create and Associate a Template Script File
  3. Define and Use a Json Object as a Data Model to be Used with the Template
  4. Discover and Play Around with the Template Refresh Options
  5. Define and Use Events
  6. Declare Bindings to Automate the Refresh Mechanism
  7. Discover a Couple of Widgets: {@aria:CheckBox} and {@aria:TextField}

Module Controllers, Application Development

  1. Use a Given Set of Provided Structured JSON Data
  2. Write Some Templates to Nicely Display the Data
  3. Create a Module Controller File with its Associated Interface
  4. Load a Template Associated with a Module Controller
  5. Create, Load and Manage a Sub Module Controller
  6. Play with IO Filters
